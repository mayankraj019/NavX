# NavX - Premium Developer Portfolio

NavX is a modern, high-performance developer portfolio website designed with React, TailwindCSS, and Framer Motion. It features interactive micro-interactions, dark mode transitions, a project showcase timeline, and a fully connected contact form.

## 🚀 Live Demo

- **Deployment URL**: [https://navigation-t3.vercel.app](https://navigation-t3.vercel.app)
- **Vercel Mirror**: [https://navigation-t3-c0veiuso7-mayankraj019s-projects.vercel.app](https://navigation-t3-c0veiuso7-mayankraj019s-projects.vercel.app)

---

## ✨ Key Features

- **Dynamic Design & Aesthetics**: Fluid layouts with HSL tailored color schemes, sleek dark modes, and modern typography using the Inter font family.
- **Micro-Animations**: Smooth physics-based transitions, hover cards, and interactive elements powered by Framer Motion.
- **Interactive Project Timeline**: Scroll-triggered journey mapping key achievements and milestones.
- **Functional Contact Form**: Direct-to-email form submission powered by Web3Forms with loader/spinner states and validator prompts.
- **Fully Responsive**: Pixel-perfect scaling optimized for viewport dimensions ranging from mobile screen sizes up to 4K.

---

## 🛠️ Tech Stack

- **Core**: React 19, JavaScript (ES6+)
- **Build Tool**: Vite
- **Styling**: TailwindCSS, CSS Variables
- **Animations**: Framer Motion
- **Icons**: React Icons (Feather Set)
- **Form Delivery**: Web3Forms API

## ⚙️ Local Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/mayankraj019/NavX.git
   cd NavX
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root folder of the project:
   ```env
   VITE_WEB3FORMS_ACCESS_KEY=your_web3forms_key_here
   ```
   *(Get your free key instantly at [web3forms.com](https://web3forms.com))*

4. **Run Dev Server**:
   ```bash
   npm run dev
   ```

---

## 📦 Building for Production

Compile and optimize the assets:
```bash
npm run build
```
The production-ready assets will be generated in the `dist` directory.
